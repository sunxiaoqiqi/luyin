## 8. 页面流转关系

```mermaid
flowchart TD
  A["打开应用"] --> B["主录制页"]
  B --> C["普通录制"]
  B --> D["多段录制"]
  C --> E["普通录制完成页"]
  E --> B
  D --> F["多段素材面板"]
  F --> G["片段预览与裁剪"]
  F --> H["合并导出"]
  F --> I["生成剪映草稿"]
  G --> F
  H --> F
  I --> F
  B --> J["设置页"]
  J --> B
```
